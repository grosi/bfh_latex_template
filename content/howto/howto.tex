    Eine kurze Installationsanleitung zu \LaTeX unter Windows.
    %
    %
    %
    \section{Miktex}\label{s:miktex}
        Miktex ist eine Tex-Distribution für Windows.
        %
        %
        \subsection{Installation}
            \begin{enumerate}
                \item Den passenden Net-Installer von der Projekt-Homepage herunterladen: \url{http://miktex.org/download}
                \image{content/howto/image/miktex_1}{scale=.35}{htbp}[Installations-Datei wählen]
                %
                \item Installer starten und als Installationsoption \textsf{MikTex Downloading}, sowie \textsf{Complete} angeben.\footnote{Achtung: Kann über 1GB Speicher benötigen} Anschliessend werden die verschiedenen \LaTeX -Pakete heruntergeladen.
                %
                \item Nach dem Download den Installer nochmals starten und dieses Mal \textsl{Install MikTex} anwählen. MikTex wird nun installiert.
            \end{enumerate}
