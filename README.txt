Dies ist das root Verzeichnis der vorliegenden LaTex-Vorlage
Es beinhaltet folgende Elemente:
- report.tex -> Haupt-Tex-Datei
- acronyms.tex -> Abkürtzungsverzeichnis
- glossary.tex -> Glossarverzeichnis
- bibliography.bib -> Literaturverzeichnis (BibTex)
- functions -> Eigene LaTex-Funktionen für mehr Konfort
- packages -> Verwendete Pakete
- listings -> Einstellungen zum Listing Paket
- colors -> Farbdefinitionen
- thorems -> Theoremdefinitionen
